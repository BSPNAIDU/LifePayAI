# LifePay AI

## Project Title
LifePay AI: An Intelligent Digital Payment Platform for Financial Wellness and Community Support

## Tagline
Pay Smart. Save Smart. Support Society.

## Description
LifePay AI is a next-generation digital payment and financial wellness platform developed using React Native CLI, Node.js, MongoDB, Docker, Jenkins, and AWS. The application combines a smart wallet, AI-based financial insights, community support, and a complete DevOps pipeline.

## Modules

### 1. Authentication
- Register
- Login
- Profile

### 2. Smart Wallet
- Wallet Balance
- Send Money (Simulation)
- Receive Money (Simulation)
- Transaction History

### 3. AI Financial Assistant
- Budget Planning
- Expense Tracking
- Savings Suggestions

### 4. Community Support
- Student Donations
- Medical Emergency Fund
- Community Wallet

### 5. Admin Dashboard
- User Management
- Reports
- Analytics
